RStudio

IDE (Integrated Development Environment) terbaik untuk bahasa pemrograman R. Digunakan untuk analisis statistik, visualisasi data, dan data mining.

Gratis Windows / Mac / Linux Butuh R terlebih dahulu

Kebutuhan Sistem

Sistem Operasi
Windows 10/11, macOS, Linux
RAM
Min. 4 GB (8 GB disarankan)
Storage
± 500 MB
Prasyarat
R harus diinstall lebih dulu

Langkah Instalasi

1

Download dan Install R terlebih dahulu

Kunjungi situs CRAN di cran.r-project.org. Pilih versi sesuai sistem operasimu (Windows/macOS/Linux). Download filenya, lalu jalankan installer. Klik Next hingga selesai — gunakan pengaturan default.

⚠️ R harus sudah terinstall sebelum menginstall RStudio. Tanpa R, RStudio tidak bisa berjalan.
2

Download RStudio Desktop

Buka posit.co/download/rstudio-desktop (Posit adalah pengembang resmi RStudio). Scroll ke bawah dan pilih RStudio Desktop — Free. Klik tombol download sesuai OS-mu.

3

Jalankan installer RStudio

Buka file .exe (Windows) atau .dmg (macOS) yang sudah didownload. Klik Next / Install terus sampai selesai. Tidak ada konfigurasi khusus yang perlu diubah.

4

Verifikasi instalasi

Buka RStudio dari Start Menu / Applications. Jika muncul tampilan dengan 4 panel (Console, Script, Environment, Files), instalasi berhasil! Coba ketik print("Halo Dunia") di panel Console lalu tekan Enter.

Anaconda

Platform data science berbasis Python yang sudah mencakup 1500+ paket siap pakai, Jupyter Notebook, Spyder, dan Anaconda Navigator untuk manajemen environment yang mudah.

Gratis Windows / Mac / Linux Sudah termasuk Python

Kebutuhan Sistem

Sistem Operasi
Windows 8+, macOS 10.13+, Linux
RAM
Min. 4 GB (8 GB disarankan)
Storage
± 3 GB (full install)
Prasyarat
Tidak perlu install Python sendiri

Langkah Instalasi

1

Download Anaconda Installer

Buka anaconda.com/download. Klik tombol download sesuai sistem operasimu. Ukuran file sekitar 900 MB, jadi pastikan koneksimu stabil saat mengunduh.

2

Jalankan installer

Buka file yang sudah didownload. Klik Next, setujui License Agreement, lalu saat ditanya pilih "Just Me" (bukan All Users). Biarkan folder instalasi default, jangan ubah.

⚠️ Hindari menginstall di folder yang mengandung spasi, contoh: C:\Program Files\Anaconda bisa menyebabkan error. Gunakan C:\anaconda3.
3

Centang "Add to PATH" (khusus Windows)

Di halaman Advanced Installation Options, centang opsi "Add Anaconda3 to my PATH environment variable". Meskipun ada peringatan, ini memudahkan penggunaan Anaconda lewat terminal/command prompt.

4

Selesaikan instalasi dan buka Anaconda Navigator

Klik Install dan tunggu proses selesai (±5-10 menit). Setelah selesai, cari Anaconda Navigator di Start Menu. Dari sini kamu bisa buka Jupyter Notebook, Spyder, atau tools lainnya.

5

Verifikasi dengan Jupyter Notebook

Di Anaconda Navigator, klik Launch di bawah Jupyter Notebook. Browser akan terbuka secara otomatis. Coba buat notebook baru dan ketik print("Halo Python") lalu tekan Shift+Enter.

SuperDecisions

Software pengambilan keputusan yang mengimplementasikan metode Analytic Hierarchy Process (AHP) dan Analytic Network Process (ANP) untuk pemodelan keputusan kompleks.

Gratis Windows (direkomendasikan) Butuh Java (JRE)

Kebutuhan Sistem

Sistem Operasi
Windows (paling stabil)
RAM
Min. 2 GB
Storage
± 200 MB
Prasyarat
Java Runtime Environment (JRE)

Langkah Instalasi

1

Install Java Runtime Environment (JRE)

SuperDecisions membutuhkan Java untuk berjalan. Buka adoptium.net (Eclipse Temurin) atau situs Oracle, download JRE versi 8 atau yang lebih baru. Jalankan installer Java dan ikuti proses default hingga selesai.

📌 Untuk memverifikasi Java sudah terinstall, buka Command Prompt dan ketik java -version. Jika muncul nomor versi, Java sudah siap.
2

Daftar akun di SuperDecisions

Kunjungi superdecisions.com dan klik tombol Download atau Register. Buat akun gratis menggunakan email aktifmu. Verifikasi email yang dikirimkan ke inbox-mu.

3

Download SuperDecisions

Setelah login, masuk ke halaman download dan pilih installer sesuai OS-mu. Download file installernya (biasanya berformat .exe atau .jar).

4

Jalankan installer

Buka file installer yang sudah didownload. Ikuti proses instalasi default hingga selesai. Tidak banyak konfigurasi yang perlu diubah — cukup klik Next dan Finish.

5

Login dan mulai gunakan

Buka aplikasi SuperDecisions, masukkan akun yang sudah didaftar tadi. Kamu sudah bisa membuat model keputusan AHP/ANP. Coba buka salah satu sample project yang sudah tersedia untuk melihat contoh penggunaannya.
